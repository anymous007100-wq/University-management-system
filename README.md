# University Management System (Frontend)

A clean, responsive University Management System designed for student and administrative use. This project focuses on efficient frontend data handling using vanilla JavaScript and JSON.

## 🚀 Features

### Student Portal
* Profile Management: View personal academic details and GPA.
* Notice Board: Real-time system updates and announcements.
* Course Registration: View and manage enrolled courses.
* Academic Records: Comprehensive view of grades and performance.
* Fees & Payments: Track financial status and pending dues.

### Admin Portal
* User Management: Oversee student and faculty accounts.
* Course & Department Management: Manage curriculum and departmental data.
* Result Processing: Handle academic performance and grading.
* Financial Oversight: Monitor system-wide financial records.

## 🛠️ Technical Stack
* HTML5/CSS3: Semantic structure and responsive design using Flexbox.
* JavaScript (ES6): Core logic, view management, and asynchronous data fetching.
* JSON: Local data storage representing a mock database.

## 📂 Project Structure
* `index.html`: Main entry point and UI structure.
* `style.css`: Custom styling and layout definitions.
* `app.js`: Core application logic and state management.
* `data.json`: Mock data for users, courses, and notices.
